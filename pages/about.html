About Campus Times Magazine

A collaborative project by UDSM students, showcasing the power of teamwork and modern web development

Why We Made This Project

Campus Times Magazine was created as part of a Software Engineering assignment at the University of Dar es Salaam (UDSM). The primary goal of this project is to learn and practice collaborative software development using Git and GitHub.

Through this project, we aim to develop essential skills in version control, team collaboration, code review, and modern web development practices. By building a real-world web magazine, we gain hands-on experience in creating responsive, user-friendly web applications while learning to work effectively as a development team.

Project Objectives

  • Master Git and GitHub for version control and collaboration
  • Practice team-based software development workflows
  • Build a modern, responsive web application
  • Learn best practices in HTML, CSS, and JavaScript
  • Develop effective communication and project management skills

Meet Our Team

Our team consists of dedicated Computer Science students from UDSM, working together to create this dynamic campus magazine platform. Each member has contributed their unique skills and perspectives to bring this project to life.
